显示问题

当添加多个标签时，小书匠编辑器默认是这样的：

此时网页端显示就会有问题，三个标签识别为一个:

#新建,模板,小书匠

解决方法

如果要添加多个标签，有两种方法书写，都有效：

  • 中括号包围

需要将标签以英文逗号隔开，并在最外层用中括号包围。

  • 换行书写
  • 实际显示效果：

#新建 #模板 #小书匠
