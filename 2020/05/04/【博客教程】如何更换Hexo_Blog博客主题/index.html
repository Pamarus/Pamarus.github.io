【博客教程】如何更换Hexo_Blog博客主题

发表于 2020-05-04 Edited on 2020-05-05 分类于 教程

Blog安装了NexT主题，所以博文以NexT为例

下载主题

有两种方法下载主题：克隆和下载压缩包

建议使用克隆最新版本的方式，之后的更新可以通过git pull来快速更新， 而不用再次下载压缩包替换。

下载压缩包

前往NexT GitHub项目页

选择你所需要的版本，下载 Download 区域下的 Source Code (zip)到本地。

解压所下载的压缩包至站点的 themes 目录下， 并将 解压后的文件夹名称更改为 next。

启用主题

打开博客本地文件夹的配置文件 _config.yml， 找到theme字段，并将其值从默认的 landscape 更改为 next。

验证主题

启动 Hexo 本地站点，并开启调试模式（即加上 –debug），整个命令是 hexo s --debug。

当命令行输出中提示出现：

INFO  Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to stop.

此时即可使用浏览器访问 http://localhost:4000，最终显示如下图。
