【博客教程】如何更换Hexo_Blog博客主题

发表于 2020-05-04 | 分类于 教程

Blog安装了NexT主题，所以博文以NexT为例

下载主题

有两种方法下载主题：克隆和下载压缩包

建议使用克隆最新版本的方式，之后的更新可以通过git pull来快速更新， 而不用再次下载压缩包替换。

克隆主题

cd到博客本地文件夹，

cd方法：

再输入用以下命令：

git clone https://github.com/iissnan/hexo-theme-next themes/next

格式：

（themes是本地博客文件夹名，主题名是本地展示的主题文件夹名称）

等待下载完成:

最后本地 themes 文件夹下载成功next文件。

更多主题参考Hexo官网：
https://hexo.io/themes/

下载压缩包

前往NexT GitHub项目页

选择你所需要的版本，下载 Download 区域下的 Source Code (zip)到本地。

解压所下载的压缩包至站点的 themes 目录下， 并将 解压后的文件夹名称更改为 next。

启用并验证主题

（1）打开博客本地文件夹的配置文件 _config.yml， 找到theme字段，并将其值从默认的 landscape 更改为 next。

（2）再启动 Hexo 本地站点，并开启调试模式（即加上 –debug），整个命令是 hexo s --debug。

当命令行输出中提示出现：

INFO  Hexo is running at http://0.0.0.0:4000/. Press Ctrl+C to stop.

此时即可使用浏览器访问 http://localhost:4000，检查站点是否正确运行。

（3）若博客关联了GitHub，终端界面输入hexo g -d命令，next主题便成功部署到GitHub上了。
